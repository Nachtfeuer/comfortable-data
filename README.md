# comfortable-data
Java based REST service for movies, books and similar data using jpa in its backend.

## Quickstart

```bash
./mvnw clean package
```

## Adjusted for using Maven Wrapper

```bash
mvn -N io.takari:maven:0.7.5:wrapper
```
