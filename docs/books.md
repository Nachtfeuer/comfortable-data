# Managing books

## Using Yaml

```
title: some title in your langauge
originalTitle: some title in the original language
series: title of the series if the book belongs to one
pages: 500
authors:
  - fullName: The full name of author
  - fullName: The full name of a further author
publisher:
  fullName: full name of a publisher
yearOfPublication: 2019
isbn: the isbn number
description: |
    Some multline text being usually the text describing the
    content of the book in short.
tags:
    - science-fiction
    - fantasy
rating: great
```
